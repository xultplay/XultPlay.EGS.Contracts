# XultPlay.EGS.Contracts

Shared record types defining the wire contract between the XultPlay EGS bridge,
handlers, notifier, and EGS bot bridge.

## Status

**v1.0.0 — under construction.** This package is currently being scaffolded.
Full documentation, usage examples, and a feature list will be added in Step 4g
of the build plan.

## Namespaces

- `XultPlay.EGS.Contracts.Messaging` — envelope types (CommandMessage,
  ReplyMessage, BroadcastMessage, EventRejection, etc.)
- `XultPlay.EGS.Contracts.Events` — **reserved**; ships empty in v1.0.0.
